Abstract

This project predicts customer churn in the telecom industry using machine learning. We compare Logistic Regression, Decision Trees, Gradient Boosting, Random Forest, and K-Nearest Neighbors using Accuracy, F1-Score, and AUC-ROC.

Introduction

  • Telecom industry is highly competitive; churn leads to revenue loss.
  • Challenge: Identifying at-risk customers before they leave.
  • Objective: Predict churn and support retention strategies.

Objectives

General: Build an accurate churn classification model.

  • Preprocess and explore dataset.
  • Implement and evaluate five ML algorithms.
  • Determine optimal model using Accuracy, Recall, F1-Score, and AUC-ROC.
  • Provide actionable insights for retention campaigns.

Literature Review

  • Logistic Regression for interpretability.
  • Random Forest & Gradient Boosting improve accuracy.
  • Advanced techniques like XGBoost are effective.
  • Hybrid and deep learning models enhance performance.

Methodology (CRISP-DM)

  1. Business Understanding
  2. Data Understanding
  3. Data Preparation
  4. Modeling
  5. Evaluation
  6. Deployment

Data Understanding

  • Dataset: 20 features + 1 target (Churn).
  • Data types: Nominal, Ordinal, Numeric.
  • Missing values handled using median/mode/zero imputation.
  • Irrelevant features (customerID) removed.

Data Transformation

  • Ensured correct numerical data types.
  • One-Hot Encoding for categorical variables.
  • Final dataset contains ~31 features.

Classification & Algorithms

  • Binary classification: Churn vs No Churn.
  • Algorithms: Logistic Regression, Decision Tree, Random Forest, Gradient Boosting, KNN.
  • Goal: Compare performance and select best model.

Tools & Technologies

  • Programming: Python
  • Environments: Anaconda, Google Colab, VS Code
  • Libraries: Pandas, NumPy, Matplotlib, scikit-learn

Performance Comparison

Accuracy: Logistic Regression 80.48%, Gradient Boosting 79.99%, Random Forest 79.91%, Decision Tree 79.56%, KNN 74.80%

Churn Recall: Logistic Regression 56%, Decision Tree 55%, Gradient Boosting 52%, KNN 50%, Random Forest 48%

Conclusion

  • CRISP-DM ensures a structured workflow.
  • Logistic Regression achieved best overall performance.
  • Model supports proactive customer retention.
